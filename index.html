医療機器エンジニア 日々の仕事の中で、「人が安心して使い続けられる仕組み」を考えています。

プロフィール

医療機器開発に20年以上携わっています。現場で本当に使われる仕組みを設計することを大切にしています。必要以上に複雑にせず、運用負担が少なく安全に維持できることを重視しています。

取り組んでいること

  • RFID を用いた薬剤・医療機器管理システム
  • M5Stack / BLE / センサを活用した現場向け IoT デバイス
  • Django / API 連携による運用改善システム

スタンス

「使い続けられるシステム」は技術だけでは成り立ちません。現場の動線、心理的な負担、メンテナンス性を含めて成立します。技術は人を支えるための手段であり、その本質を忘れない設計を心がけています。

趣味

  • 土地や食が変わると、人の感覚も変わるのが面白いと感じます。
  • 料理は、考えるより手が先に動く時間です。
